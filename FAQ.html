FAQ

A página FAQ se destina servir de eventuais dúvidas, e suporte para usuários que apresentam alguma dificuldade em acessar e interagir com os recursos do website.

Esta página foi projetada utilizando as linguagens HTML 5, CSS 3 e a linguagem de programção Javascript, junto do framework Bootstrap.

Para navegar no website, basta clicar entre os links do menu de navegação, que é fixo no topo de todas as guias do site. Cada guia leva a uma página diferente com seus conteúdos.

Entretanto, caso o usuário esteja navegando em uma tela de celular, basta clicar na gaveta abaixo para aparecer a lista de links.

É possível descobrir mais sobre um determinado conteúdo na página Home que esteja atrelado com um botão "Ver Mais" como no exemplo abaixo, clicando no próprio botão, o usuário será redirecionado para a guia correspondente.

Para visualizar os depoimentos na aba principal, basta clicar nas setas de navegação, de direita e esquerda, e o usuário será direcionado para a guia correspondente de depoimentos.

O Quiz Odara é um recurso lúdico feito para cativar o usuário a interagir com a página, para começar, basta o usuário clicar na alternativa da primeira pergunta, que iniciará a sequência de 7 perguntas do quiz, mostrando qual a resposta certa ao final de cada alternativa.
