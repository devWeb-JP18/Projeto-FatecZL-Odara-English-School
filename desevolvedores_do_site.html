Projeto Integrador Fatec

O Projeto Integrador é um projeto proposto pela Fatec Zona Leste para os alunes do curso de Desenvolvimento de Software Multiplataforma. Neste projeto, os alunes foram encorajados a entrar em contato com empresas e projetos na região da Zona Leste de São Paulo para desenvolver um website para a instituição escolhida.

Esta inciativa visa não só o desenvolvimento e avaliação acadêmica dos alunos, mas também vem a fomentar e contribuir para o aumento e evolução dos comércios da Zona Leste de SP, visto que a Fatec ZL é de suma importância para o crescimento da região como um todo.
